Demo of WordPress Customizer added to a plugin.

== Description ==

The plugin demonstrate WordPress Customizer added to a plugin. If you'd like to add WordPress Customizer to your plugin, please see the Plugin Customizer [documentation at GitHub](https://github.com/soderlind/plugin-customizer#plugin-customizer)

== Installation ==

You know the drill:

1. In WordPress Admin, go to `Plugins->Add New`
1. Search for `Plugin Customizer`
1. Install and Activate

or

1. Download the plugin and extract the plugin-customizer.zip
1. Upload the extracted `plugin-customizer` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

The plugin is also [available at GitHub](https://github.com/soderlind/plugin-customizer)
